# 论文图表管理器

## 功能

### 1. 图号表号校验

- 图号按章编号：图 3.1, 3.2, ..., 图 4.1, 4.2, ...
- 表号同理：表 2-1, 表 4-1, ...
- 检查：是否连续、是否有跳号、是否重复
- 修复：自动重编号，按文中出现顺序递增

### 2. 占位符管理

- `【待插入图】` 需替换为实际文件路径
- `【待插入系统截图】` 需替换为 `screenshots/platform/xxx.png`
- `【待插入图：pic/xxx】` 格式保留，等待制作完成后插入

### 3. 截图映射

- 第五章截图 `S-01` ~ `S-37` 映射到 `图 5.2` ~ `图 5.25`
- 映射表存储在 `thesis_missing_figures_tables.md`
- 检查：是否有截图文件但论文未引用、论文引用但截图文件缺失

### 4. 图题表题格式

- 图题：宋体五号居中，放在图下方。格式：`图 X.Y 图名`
- 表题：宋体五号居中，放在表上方。格式：`表 X-Y 表名`
- 图题不得写在占位框内
- 表题不得写在表格下方

### 5. 交叉引用

- 正文中"如图 X.Y 所示""见表 X-Y"的引用号与实际图号表号一致
- 修复：引用的图号表号随重编号自动更新

### 6. 生成缺失清单

- 输出 `thesis_missing_figures_tables.md`
- 列出所有缺失的图/表、所在位置、优先级、制作建议

## 执行

读取论文 Markdown，自动扫描上述问题并修复。优先处理图号表号冲突和占位符路径缺失。
