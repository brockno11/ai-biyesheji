# 论文 Markdown 结构校验

## 校验规则

### 标题层级

- 一级标题 `#` 只能用于：摘要、Abstract、目录、第X章、参考文献、附录、致谢
- 二级标题 `##` 只能用于章内一级分节（如"一、可行性分析"）
- 三级标题 `###` 只能用于二级子节（如"（一）社会可行性"）
- 标题层级不得跳级（不能 # 后直接 ###）

### 章号连续

- 七章编号连续：第一章~第七章
- 章标题格式统一：`# 第X章 标题`

### 分页标记

- 每章末尾必须有 `<div style="page-break-after: always;"></div>`
- 摘要、Abstract、目录后必须有分页标记

### 摘要/关键词

- 中文关键词 4-6 个，用中文分号分隔
- 英文 Keywords 与中文关键词数量对应
- 关键词不得包含废话（如"研究""分析"）

### 参考文献

- 编号 [1]~[N] 连续无跳号
- 每条包含作者、题名、来源、年份等基本字段
- 不得有【待补充】残留（除封面个人信息外）

### 残留检查

- 扫描全文中所有 `【待撰写】` / `【待补充】`
- 除封面（学号、姓名等）和文件头注释外，正文中不应有任何残留
- `【待填写】` 仅允许存在于封面信息行

## 执行

运行此 skill 时，读取论文 Markdown 文件，逐条验证上述规则，不符合的直接修复。
