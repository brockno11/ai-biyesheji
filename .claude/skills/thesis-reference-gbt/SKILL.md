# 参考文献 GB/T 7714 格式管理

## 功能

### 1. 编号连续检查

- 文献表编号 [1]~[N] 连续无跳号
- 正文中引用的编号必须在文献表中存在
- 文献表中的条目必须在正文中被引用（或合理保留）

### 2. 格式完整性

每条参考文献必须包含：

| 文献类型 | 必含字段 |
|----------|---------|
| 期刊 [J] | 作者. 题名[J]. 刊名, 年份, 卷(期): 页码. |
| 学位论文 [D] | 作者. 题名[D]. 出版地: 学校, 年份. |
| 会议 [C] | 作者. 题名[C]//会议名称, 年份: 页码. |

### 3. 残缺标记

- 缺失期刊名、卷期、页码等信息的，标为 `【待补充：XXX】`
- 无法确认真实性的文献，标为 `【待核实】`

### 4. 常见问题修复

- 英文文献作者大小写不统一
- 卷期页码缺失
- DOI 格式错误
- 中英文混排的标点符号不一致

### 5. 作者年份引用格式

- 正文中"Author (Year)"或"Author 等人 (Year)"格式
- 与参考文献表编号对应
- 正文编号 [1] [2] 按首次出现顺序

## 执行

读取论文参考文献章节，逐条校验格式。对残缺条目标注 `【待补充】`；对格式不规范的自动修正；对无法确认真实性的给出提示。
