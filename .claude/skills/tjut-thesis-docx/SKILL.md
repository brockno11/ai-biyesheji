# 天津工业大学本科论文 Word 排版 Skill

## 目标

把 thesis.md 转换为符合《天津工业大学本科毕业论文格式模板（2024修订版）》的 docx 文件。

禁止只做普通 Markdown 转 Word。必须：
1. 读取学校模板 docx。
2. 读取前辈参考 PDF。
3. 使用 python-docx + lxml 生成 Word。
4. 生成后进行结构审查和渲染审查。
5. 输出 docx 和格式审查报告。

## 输入

- thesis.md
- template/本科毕业论文格式模板.docx
- reference/前辈参考论文.pdf
- assets/figures/
- README.md
- DOCS.md
- COURSE_METHODOLOGY.md
- SKILLS.md

## 输出

- output/张朔_本科毕业论文_排版稿.docx
- output/格式审查报告.md
- output/缺失图片和表格清单.md

## 核心格式规则

1. 封面：题目、学号、姓名、专业、学院、指导教师、职称、完成日期。
2. 中文摘要：“摘  要”三号黑体居中，正文小四宋体。
3. 英文摘要：Abstract 三号 Times New Roman 加粗居中，正文小四 Times New Roman。
4. 目录：“目  录”三号黑体居中，目录项由 Word 标题样式生成。
5. 章标题：第一章 绪论，小三黑体，居中，段前段后 0.5 行。
6. 二级标题：一、研究背景，四号黑体，两端对齐，悬挂缩进 0.97 cm。
7. 三级及以下标题：（一）研究背景，小四黑体，悬挂缩进 0.94 cm。
8. 正文：小四，中文宋体，英文 Times New Roman，1.5 倍行距，首行缩进 0.84 cm。
9. 每章另起一页。
10. 表格：三线表，表题在表上方，宋体五号居中，编号连续。
11. 图片：居中，比例不变形，图题在图下方，宋体五号居中，编号连续。
12. 参考文献：另起一页，按引用顺序排列，五号宋体 / Times New Roman。
13. 附录、致谢另起一页。
14. 页眉：天津工业大学 2026 届本科生毕业设计（论文）。
15. 正文页码从正文开始。

## 工作流程

### P0：结构审查
先检查 thesis.md，不生成 Word：
- 标题层级是否规范
- 图表编号是否连续
- 图片路径是否存在
- 表格是否过宽
- 参考文献是否缺失或格式混乱
- 是否存在“待补充”“TODO”“图待插入”等内容

### P1：生成脚本
创建 scripts/build_thesis_docx.py：
- 解析 Markdown
- 建立 Word 样式
- 套用模板页眉、页脚、页面边距
- 插入封面、摘要、Abstract、目录、正文、参考文献、附录、致谢
- 插入图片和图题
- 插入表格和表题
- 处理三线表
- 处理分页

### P2：生成 Word
运行脚本，输出 output/张朔_本科毕业论文_排版稿.docx。

### P3：自动审查
创建并运行 scripts/audit_thesis_docx.py：
- 检查 styles.xml
- 检查 document.xml
- 检查页眉页脚
- 检查图片、表格、标题、参考文献
- 输出格式审查报告

### P4：渲染审查
将 docx 导出为 PDF 或页面图片，检查：
- 页面是否溢出
- 表格是否断裂
- 图片是否变形
- 标题是否跑版
- 页眉页码是否正确
- 图题表题位置是否正确
