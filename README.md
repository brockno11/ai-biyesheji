# 基于 AI 赋能的机器学习算法教学平台

## 项目简介

面向机器学习初学者的交互式 Web 教学平台，融合 **AI 助教**、**算法可视化** 和 **在线代码练习** 三大核心功能，帮助初学者系统学习机器学习核心算法。

## 技术栈

| 技术 | 用途 |
|------|------|
| React 18 | UI 框架 |
| TypeScript | 类型安全 |
| Vite | 构建工具 |
| Tailwind CSS | 样式方案 |
| React Router v6 | 前端路由 |
| Monaco Editor | 代码编辑器 |
| ECharts / echarts-for-react | 数据可视化 |
| localStorage | 本地数据持久化 |
| OpenAI-compatible API (可选) | AI 助教后端 |

## 功能模块

### 1. 算法课程学习
- 3 个核心算法：线性回归、KNN、决策树
- 每个算法包含：简介、公式、步骤、优缺点、适用场景、Python 代码示例
- B 站教学视频嵌入
- 交互式可视化组件

### 2. AI 助教
- 解释概念：用通俗语言讲解算法原理
- 诊断代码：分析学生代码并给出改进建议
- 学习建议：根据进度推荐下一步学习内容
- 题目生成：根据当前算法生成测验题
- 支持真实 API（设置环境变量）和本地 Mock（默认）

### 3. 算法可视化
- **线性回归**：散点图 + 拟合直线 + Loss 曲线，可调参数（学习率、迭代次数、噪声）
- **KNN**：二维分类散点图 + K 值调节 + 最近邻居高亮 + 可点击移动测试点
- **决策树**：树结构可视化 + 最大深度调节 + 样本分类路径追踪

### 4. 代码练习
- Monaco Editor 在线 Python 编辑器
- 规则检查引擎（检测关键 API 使用情况）
- 得分评估和详细改进建议
- 练习记录自动保存

### 5. 题目测验
- 每个算法 5 道选择题
- 涵盖核心概念、公式、API 使用
- 自动评分和答案解析

### 6. 学习进度
- 总体进度概览
- 各算法完成状态
- 练习次数和最高分
- 最近活动记录
- 学习路径推荐

## 安装方式

```bash
# 1. 克隆项目
cd demo-biyesheji

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

## 运行方式

### 开发模式
```bash
npm run dev
# 访问 http://localhost:3000
```

### 生产构建
```bash
npm run build
npm run preview
```

### 开启真实 AI API（可选）

```bash
# Windows PowerShell
$env:VITE_AI_API_KEY="your-api-key"
$env:VITE_AI_BASE_URL="https://api.openai.com/v1"  # 或其他兼容 API
$env:VITE_AI_MODEL="gpt-3.5-turbo"

# Linux / macOS
export VITE_AI_API_KEY="your-api-key"
export VITE_AI_BASE_URL="https://api.openai.com/v1"
export VITE_AI_MODEL="gpt-3.5-turbo"

npm run dev
```

不设置环境变量时，AI 助教使用预设 Mock 数据，不影响演示。

## 项目亮点

1. **零依赖即可演示**：不配置任何 API Key 也能完整运行和演示
2. **交互式可视化**：通过 slider 调节参数，直观感受算法变化
3. **AI 全流程陪伴**：从概念解释到代码诊断，AI 助教贯穿学习全过程
4. **本地存储**：学习记录保存在浏览器，无需后端数据库
5. **规则化代码检查**：不依赖 Python 运行时，通过关键词规则即时反馈
6. **答辩友好**：UI 设计精美，功能完整，适合毕业答辩现场演示

## 毕业设计说明

本项目设计目标是构建一个 **可运行、可演示、适合答辩展示** 的机器学习教学 MVP。

### 设计原则
- **前端优先**：所有数据静态化，无需后端服务
- **交互第一**：重视可视化交互和用户体验
- **AI 赋能**：AI 助教是核心差异化功能
- **学习闭环**：课程学习 → 代码练习 → 测验验证 → 进度追踪

### 后续可扩展方向
- 对接真实 Python 沙箱（Pyodide），实现在线代码执行
- 接入大模型微调（Fine-tuning），提升 AI 助教回答质量
- 增加更多算法（SVM、随机森林、神经网络等）
- 后端服务化（用户系统、数据持久化、学习分析）
- 增加视频录制/直播授课功能
- 添加学习社区和讨论区
- 移动端适配（响应式优化）
- 国际化（中英文切换）
